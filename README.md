# StudGuide УрФУ — База знаний для первокурсников (MVP)

Проект представляет собой единый веб-ресурс (базу знаний) для студентов первого курса. Стек: Python, Django, PostgreSQL, Tailwind CSS + HTMX.

## 🚀 Как запустить проект локально

### 1. Подготовка базы данных

Убедитесь, что у вас установлен PostgreSQL. Создайте пустую базу данных с названием `studguide`.

### 2. Настройка окружения и установка зависимостей

Склонируйте репозиторий и перейдите в папку проекта. Откройте терминал и выполните команды:

```bash
# Создание и активация виртуального окружения
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Mac/Linux:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

### 3. Настройка переменных окружения (.env)

Переименуйте `.env.example` на `.env` и вставьте настройки подключения к вашей локальной базе данных:


### 4. Создание таблиц и запуск

```bash
# Применяем миграции
python manage.py migrate

# Создаем аккаунт редактора (Суперпользователя)
python manage.py createsuperuser

# Запускаем локальный сервер
python manage.py runserver
```

Проект доступен по адресу: **<http://127.0.0.1:8000/>**

## 📝 Как управлять контентом (Инструкция редактора)

1. **Вход в систему:** Нажмите кнопку «Войти» в шапке сайта или перейдите по адресу `http://127.0.0.1:8000/admin/`. Используйте данные суперпользователя, созданного на этапе установки.
2. **Добавление категорий:** В боковом меню выберите «Категории» ➔ «Добавить». Введите название (например: *Заселение, Быт в общежитии*). Поле "URL Slug" заполнится автоматически. Нажмите «Сохранить».
3. **Добавление статей:** Выберите «Статьи» ➔ «Добавить».
   - Напишите заголовок и выберите категорию.
   - Используйте встроенный визуальный редактор (TinyMCE) для оформления текста (списки, абзацы, заголовки).
   - Выберите статус `Опубликовано`, чтобы статья появилась на сайте. По умолчанию новые статьи сохраняются как `Черновик`.
   - Нажмите «Сохранить».

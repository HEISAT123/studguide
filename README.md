# StudGuide УрФУ — База знаний для первокурсников (MVP)

Проект представляет собой единый веб-ресурс (базу знаний) для студентов первого курса. 
**Стек:** Python, Django, PostgreSQL, Tailwind CSS + HTMX, Docker.

---

## 🚀 Как запустить проект локально


### 🐳 Способ 1: Запуск через Docker (Рекомендуемый)


**1. Настройка переменных окружения:**  
Переименуйте `.env.example` на `.env` и вставьте настройки подключения к вашей локальной БД:
```env
SECRET_KEY=django-insecure-your-secret-key
DEBUG=True
DB_NAME=studguide
DB_USER=postgres
DB_PASSWORD=пароль_от_базы_данных_на_вашем_пк
DB_HOST=db
DB_PORT=5432
```

**2. Запуск контейнеров:**
Выполните команду для сборки и запуска архитектуры (База данных + Nginx + Django):
```bash
docker compose up -d --build
```

**3. Создание администратора:**
Миграции и сборка статики выполнятся автоматически. Вам остается только создать аккаунт редактора:
```bash
docker-compose exec web python manage.py createsuperuser
```

Проект доступен по адресу: **http://localhost**

---

### 💻 Способ 2: Классический запуск (без Docker)

**1. Подготовка базы данных:**  
В вашей локальной СУБД PostgreSQL создайте пустую базу данных с названием `studguide`.

**2. Настройка виртуального окружения:**
```bash
python -m venv venv

# Активация для Windows:
venv\Scripts\activate
# Активация для Mac/Linux:
source venv/bin/activate
```

**3. Установка зависимостей:**
```bash
pip install -r requirements.txt
```

**4. Настройка переменных окружения (.env):**  
Переименуйте `.env.example` на `.env` и вставьте настройки подключения к вашей локальной БД:
```env
SECRET_KEY=django-insecure-your-secret-key
DEBUG=True
DB_NAME=studguide
DB_USER=postgres
DB_PASSWORD=пароль_от_базы_данных_на_вашем_пк
DB_HOST=db
DB_PORT=5432
```

**5. Применение миграций и запуск:**
```bash
# Создаем таблицы в БД
python manage.py migrate

# Собираем статические файлы
python manage.py collectstatic

# Создаем аккаунт редактора
python manage.py createsuperuser

# Запускаем локальный сервер
python manage.py runserver
```

Проект доступен по адресу: **http://localhost:8000/**

## 📝 Как управлять контентом (Инструкция для Редактора)

1. **Вход в систему:** 
   Нажмите кнопку «Войти» в шапке сайта. Используйте логин и пароль суперпользователя, созданного на этапе установки.
2. **Добавление категорий:** 
   В боковом меню выберите «Категории» ➔ «Добавить». Введите название (например: *Заселение, Быт в общежитии*). Поле "URL Slug" заполнится автоматически. Нажмите «Сохранить».
3. **Добавление статей:** 
   Выберите «Статьи» ➔ «Добавить».
   - Напишите заголовок и выберите категорию.
   - Используйте встроенный визуальный редактор для оформления текста (поддерживаются списки, абзацы, заголовки, вставка ссылок и картинок).
   - Выберите статус `Опубликовано`, чтобы статья сразу появилась на сайте. По умолчанию новые статьи сохраняются как `Черновик`.
   - Нажмите «Сохранить». Автор статьи будет зафиксирован автоматически.
